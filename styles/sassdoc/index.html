    mvl-respond-to

    @mixin mvl-respond-to($breakpoint) { ... }

    Description

    Responsive manager

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $breakpoint

    Breakpoint

    		String—none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    Input (SCSS)

    $mvl-breakpoints: (
  'medium': (min-width: 800px),
  'large': (min-width: 1000px),
);

@include mvl-respond-to('medium') {
    /* media query specific CSS rules */
}

    Output (CSS)

    @media only screen and (min-width: 800px) {
    /* media query specific CSS rules */
}

    Throws

    • No value found for #{$breakpoint}.

    Requires

    • [variable] mvl-breakpoints

    Author

    • Hugo Giraudel

    mvl-font-size-line-height

    @mixin mvl-font-size-line-height($font-size) { ... }

    Description

    Converts font size (input in px) to rem and outputs it. Also outputs corresponding (i. e. a multiple of $mvl-base-line-height) line-height.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $font-size

    font size in px

    		Number—none

    Example

    Input (SCSS)

    @include mvl-font-size-line-height(20px);

    Output (CSS)

    font-size: 1.1875rem;
line-height: 1.31579;

    Throws

    • mvl-font-size-line-height only supports px inputs.

    Requires

    • [variable] mvl-base-font-size-screen
    • [variable] mvl-base-line-height
