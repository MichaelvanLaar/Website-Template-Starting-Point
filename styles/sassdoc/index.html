  main-content-width: (
    min-width: (
      (size(column-width, px) * map-get($max-columns, main-content)) +
        (
          size(inner-gutter-width, px) *
            (map-get($max-columns, main-content) - 1)
        ) + (size(outer-gutter-width, px) * 2)
    )
  ),
  full-width: (
    min-width: (
      (size(column-width, px) * 12) + (size(inner-gutter-width, px) * 11) +
        (size(outer-gutter-width, px) * 2) + 50px
    )
  ),
  full-width--scale-1: (
    min-width:
      floor(
        (
            (size(column-width, px) * 12) + (size(inner-gutter-width, px) * 11) +
              (size(outer-gutter-width, px) * 2) + 150px
          ) * 1.0625
      )
  ),
  full-width--scale-2: (
    min-width:
      floor(
        (
            (size(column-width, px) * 12) + (size(inner-gutter-width, px) * 11) +
              (size(outer-gutter-width, px) * 2) + 250px
          ) * 1.125
      )
  ),
  full-width--scale-3: (
    min-width:

    respond-to

    @mixin respond-to($breakpoint) { ... }

    Description

    Responsive manager

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $breakpoint

    Named breakpoint from $breakpoints map

    		String— none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    Input (SCSS)

    $breakpoints: (
  medium: (min-width: 800px),
  large: (min-width: 1000px),
);

@include respond-to(medium) {
  /* media query specific CSS rules */
}

    Output (CSS)

    @media only screen and (min-width: 800px) {
  /* media query specific CSS rules */
}

    Throws

    • No value found for #{$breakpoint}.

    Requires

    • [variable] breakpoints

    Author

    • Hugo Giraudel

);

    Description

    Basic sizes in px, with reference to the non-scaled versions of the layout

    Type

    Map

    Map structure

    Map key NameMap key DescriptionMap key TypeMap key Value
    column-width

    Width (in px) of one column of the 12-column layout grid

    		Number48px
    inner-gutter-width

    Width (in px) of the gutter between columns of the 12-column layout grid

    		Number24px
    outer-gutter-width

    Width (in px) of the gutter to the left and to the right of the 12-column layout grid (= horizontal page margins)

    		Number24px
    base-font-size

    Font size (in px) for basic copy text

    		Number16px
    base-line-height

    Line height (in px) for basic copy text

    		Number24px
    base-margin

    Top and bottom margin (in px) for basic copy text paragraphs

    		Number24px
    base-indent

    Indent (in px) for basic lists and blockquotes

    		Number35px
    site-logo-width

    Original width (in px) of the website’s logo (used for aspect ratio calculation)

    		Number— none
    site-logo-height

    Original height (in px) of the website’s logo (used for aspect ratio calculation)

    		Number— none
    header-height

    Height (in px) of the fixed header bar

    		Number48px
    navigation-toggle-width

    Width (in px) of the hamburger icon to slide out the main navigation menu

    		Number24px
    off-canvas-menu-width
